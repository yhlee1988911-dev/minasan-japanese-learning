# Minasan PWA 架构

## 学习路径

`课程 -> 词汇 -> 短句 -> 本课练习`

## 自由练习

- 单词听写
- 中译日
- 短句填空

## 数据边界

- `data/ci.xlsx`：原始只读数据，不由 App 直接读取。
- `src/domain`：稳定的领域模型。
- `src/data/generated`：由 `ci.xlsx` 生成的版本化词汇、课程索引与质量报告。
- `src/data/catalog.ts`：App 使用的统一数据入口；短句原型与正式词汇在此汇合。
- `src/storage`：用户进度、错题、收藏和记忆曲线的本地存储入口。
- `src/pages`：课程、课时和练习页面。

## 正式数据管线

1. 读取 Excel，但不修改源文件。
2. 映射并校验词汇 ID、假名、声调、词性、释义和课时。
3. 保留重复词的课程出现关系，不直接删除。
4. 由人工审核短句及答案集合。
5. 输出 `public/data/catalog.json` 和按课时拆分的数据文件。
6. 构建时校验引用完整性，PWA 缓存已发布的数据版本。

执行 `npm run data:import` 可重新读取 Excel 并生成 1–50 课数据。该命令不会修改源文件。

## 后续模块

- `mistakes`：错题记录，包含模式、词汇/短句 ID 和错误答案。
- `favorites`：收藏词汇和短句。
- `review`：基于间隔重复的复习队列。
- `sync`：未来账号同步；当前保持 local-first。
